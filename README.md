# JOGLFAddon-GaussianBlur

JOGLFAddon that applies Gaussian blur to screens

# Installation

```xml
<dependency>
    <groupId>com.github.dabasan</groupId>
    <artifactId>joglfaddon-gaussianblur</artifactId>
    <version>1.0.0</version>
</dependency>
```
